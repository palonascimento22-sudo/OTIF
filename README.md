Interactive OTIF (On Time In Full) dashboard built with Power BI for logistics and delivery performance analysis.

Objetivo: analisar nível de serviço OTIF.

KPIs: OTIF %, entregas no prazo, entregas completas, atrasos, performance por cliente/região/transportadora.

Ferramentas: Power BI, DAX, Power Query.

Insights: identificar gargalos logísticos e oportunidades de melhoria.
